Este projeto foi desenvolvido em Java utilizando organização por pacotes e deve ser executado via linha de comando (terminal), sem dependência de IDE.
1. Requisitos:
Java JDK instalado e configurado nas variáveis de ambiente.

2. Como Compilar:
Abra o terminal na pasta raiz do projeto e execute o comando:
cd N1_JAVA_POO
javac biblioteca/*.java

3. Como Executar:
Após a compilação, execute o sistema com o comando:
java biblioteca.Main
	
(Nota: O comando de execução deve ser feito a partir da pasta raiz, referenciando o pacote 'biblioteca' seguido da classe principal 'Main')
